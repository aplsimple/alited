 In the templates, the following wildcards are used:

    <b>%U</b> - user's name
    <b>%u</b> - user's login
    <b>%m</b> - user's mail
    <b>%w</b> - user's web site
    <b>%d</b> - date (default format is %D)
    <b>%t</b> - time (default format is %T)
    <b>%F</b> - current file's name
    <b>%f</b> - current file's tail name
    <b>%n</b> - current file's root name (namespace)
    <b>%p</b> - current procedure/method's name
    <b>%a</b> - current procedure/method's arguments

 Some of them can be customized in this dialogue.

 <b>List of Templates</b> button shows the templates.
