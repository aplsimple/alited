 Use buttons on the right to add/change/delete a template.

 Use <r>Add</r> and <r>Change</r> buttons after filling template's data.

 Use <r>Import</r> button to fetch the new templates from an external
 alited.ini (mostly of the alited installation).

 Choose or erase a template's <b>hot keys</b> from the dropdown combobox.

 In a template's text, set <b>the cursor</b> where it should be at inserting.

 In the text, use the following wildcards:

    <b>%d</b> - date
    <b>%t</b> - time
    <b>%u</b> - user's login as set in "Preferences"
    <b>%U</b> - user's name as set in "Preferences"
    <b>%m</b> - user's mail as set in "Preferences"
    <b>%w</b> - user's web site as set in "Preferences"
    <b>%F</b> - current file's name
    <b>%f</b> - current file's tail name
    <b>%n</b> - current file's root name (namespace)
    <b>%p</b> - current procedure/method's name
    <b>%a</b> - current procedure/method's arguments
    <b>%A</b> - alited's directory
    <b>%M</b> - e_menu menus directory (good for helper files)
