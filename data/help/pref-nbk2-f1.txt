  Choose a font size for the edited texts.

  Set at need the following options for the texts:

    "Space above lines" is an additional space above each text line
    "Space below lines" is an additional space below each text line
    "Space between wraps" is a space between lines that are wrapped

  The following options define the bar tab's view:

    "Tab bar label's length" - a restriction for long file names
    "Tab bar tip's length" - a restriction for long file lists
