 The <b>Tkcon</b> options define the appearance of Tkcon's window.

 You can set your own colors of Tkcon.

 Use <b>Default</b> button to set default light colors.
 Use <b>Default 2</b> button to set default black colors.

 Other options define the size of Tkcon's window.

 The geometry is set as <b>+X+Y</b> meaning x- and y-coordinates of
 Tkcon's window. For <b>-geometry {}</b>, it is shown in the center
 of the screen.

 Use <b>Test</b> button to test your options of Tkcon.
